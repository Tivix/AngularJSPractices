Filter Example

Here's an example of how AngularJS filters work. I've created a date object. If I present it in its raw format, we get {{now}}. If I apply a filter, we get {{now | date : "MM/dd/yyyy"}}. Similarly, with currency, I might have a value like {{price}}. If I wanted to properly format it, I could apply the currency filter and get {{price | currency : "$"}} instead.
